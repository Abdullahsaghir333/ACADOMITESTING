--- GEMINI API (Text Extraction & Transcription Only) ---

File: backend/src/services/llm/router.ts
  • extractTextFromImage()        → Extracts text from uploaded images (multimodal vision)
  • transcribeAudioBuffer()       → Transcribes uploaded audio to text (multimodal audio)

File: backend/src/routes/uploads.ts
  • Calls extractTextFromImage() for image uploads
  • Calls transcribeAudio() for audio uploads

File: backend/src/routes/tutor.ts
  • Calls transcribeAudio() to transcribe student's spoken question (POST /sessions/:id/ask)

File: backend/src/routes/roleReversal.ts
  • Calls transcribeAudio() to transcribe student's teaching recording (POST /evaluate)

File: backend/src/routes/tutorGroup.ts
  • Calls transcribeAudio() to transcribe group member's spoken question (POST /:id/ask)

File: python/services/podcast/podcast_logic.py
  • generate_script_from_gemini() → Generates podcast dialogue script (uses google.generativeai SDK)


--- LLAMA 2 (via Ollama — Tutor Slides, Q&A, ELI5) ---

Configured in: backend/src/services/llm/config.ts
  • ollamaModelTutor() → returns "llama2" (env: OLLAMA_MODEL_TUTOR)

Called via: backend/src/services/llm/router.ts
  • completeTutorPrompt() → routes to ollamaChat(ollamaModelTutor(), prompt)

Used in: backend/src/services/gemini.ts
  • generateTutorSlidesAndScripts()   → Generates 6–14 slide decks with scripts (line 377)
  • answerTutorQuestion()             → Answers student Q&A during session (line 419)
  • generateTutorSlideEli5Script()    → Generates ELI5 simplified script (line 447)

Used in: backend/src/routes/tutor.ts
  • POST /sessions           → Creates tutor session, calls generateTutorSlidesAndScripts()
  • POST /sessions/:id/ask   → Calls answerTutorQuestion() after transcription
  • POST /sessions/:id/slides/:idx/eli5 → Calls generateTutorSlideEli5Script()

Used in: backend/src/routes/tutorGroup.ts
  • POST /:id/ask            → Calls answerTutorQuestion() for group Q&A
