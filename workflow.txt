================================================================================
              ACADOMI — FEATURE WORKFLOW DIAGRAMS (for image generation)
================================================================================

Use each section below as the content/layout guide for generating a workflow
diagram image in the same visual style as the Role Reversal diagram (clean
boxes, labeled arrows, icons for user/API/model/UI, left-to-right flow).

NOTE:  Gemini API is used ONLY for text extraction & audio transcription.
       All inference / generation tasks use local Ollama models:
         • Llama 2         → Tutor slides, Q&A, ELI5
         • Phi-3           → Role-reversal eval, Cheat sheets
         • Phi-3 Mini      → Study notes, Bookmarks, Recap
         • LLaVA           → Image text extraction (vision)
       TTS is handled by Edge TTS (tutor) and gTTS (podcast).
       Focus monitoring uses MediaPipe FaceMesh (computer vision).

================================================================================
  STYLE GUIDE (for all 4 diagrams — match the Role Reversal reference image)
================================================================================

  BACKGROUND:      Light gray (#F2F2F2)
  BOX STYLE:       White rounded rectangles with thin gray border
  GEMINI BOXES:    Light blue tint background, sparkle icon (★) for Gemini
                   (used ONLY for Text Extraction and Transcription)
  OLLAMA BOXES:    White with brain icon (🧠) for local model processing
                   Label clearly: "Ollama Model (Llama 2)" or "(Phi-3)" etc.
  TTS BOXES:       White with speaker icon for Edge TTS
  UI BOXES:        Dark header bar, white body, list items inside
  ICONS:           Flat, minimal, single-color (black or dark gray)
  ARROWS:          Red/coral colored directional arrows (like reference image)
  FONTS:           Clean sans-serif (e.g., Inter, Roboto)
  TITLE:           Bold, centered at top, large font
  SUB-ITEMS:       Listed inside boxes as indented bullet-style text
  JSON ICON:       Curly braces { } symbol
  DOCUMENT ICON:   Page with lines symbol
  AUDIO ICON:      Speaker / sound wave symbol

  KEY DISTINCTION:
    • Gemini (sparkle icon, blue tint) = Text Extraction & Transcription ONLY
    • Ollama (brain icon, white)       = All generation / inference tasks
    • Edge TTS (speaker icon)          = All audio synthesis
